### Docker Images
10. **List Docker images**
    ```bash
    docker images
    ```

11. **Pull an image from Docker Hub**
    ```bash
    docker pull <image>
    ```

12. **Remove an image**
    ```bash
    docker rmi <image_id>
    ```

13. **Build an image from a Dockerfile**
    ```bash
    docker build -t <image_name> .
    ```
